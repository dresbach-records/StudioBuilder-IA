GitHub Authentication

Connect your GitHub account to enable AI-powered code generation, automatic repository syncing, and seamless deployment workflows directly from StudioBuilder IDE.

info Setup Guide

01

Go to GitHub Settings

Navigate to your GitHub account settings and locate Developer Settings at the bottom left.

02

Generate Token

Select Personal access tokens and choose "Tokens (classic)" or "Fine-grained tokens".

03

Set Permissions

Ensure you select 'repo' and 'workflow' scopes to allow StudioBuilder to sync your code.

04

Paste & Connect

Copy the generated token and paste it into the secure field in the configuration panel.
